<h1>Beam-Column Joint Shear Predictor</h1>

<p>
This repository contains the trained machine learning models, dataset, and source code developed for the published study:
</p>

<p>
<em>Prediction of Beam-Column Joint Shear Strength Using Machine Learning and a Proposed Design Equation.</em><br>
DOI: https://doi.org/10.54878/ay1rt369
</p>

<p>
The study presents machine learning-based models for predicting the shear strength of reinforced concrete beam-column joints. Gradient Boosting, Random Forest, and a Stacked Ensemble model were developed and evaluated using an experimental database. In addition, a practical design equation was proposed to provide an interpretable and reliable alternative for engineering applications.
</p>

<h2>Repository Contents</h2>
<ul>
  <li>Trained machine learning models (Gradient Boosting, Random Forest, and Stacked Ensemble)</li>
  <li>Experimental database used for model development and validation</li>
  <li>Data preprocessing and model training scripts</li>
  <li>Feature importance and SHAP analysis</li>
  <li>Performance evaluation and comparison with ACI provisions</li>
  <li>Implementation of the proposed design equation</li>
</ul>

<p>
This repository accompanies the published article and is intended to support reproducible research and the application of machine learning techniques in reinforced concrete structural engineering.
</p>
