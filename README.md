<h1>Beam-Column Joint Shear Predictor</h1>

<p>
This repository contains the trained models, dataset, and source code developed for the study:
</p>

<p>
<strong>“Prediction of Beam-Column Joint Shear Strength Using Machine Learning and a Proposed Design Equation.”</strong>
</p>

<p>
The project implements Gradient Boosting, Random Forest, and a Stacked Ensemble of machine learning models to accurately predict reinforced concrete joint shear strength. It also includes the proposed empirical equation and feature analysis to provide interpretable insights for practical design and retrofitting applications.
</p>

<h2>Contents</h2>
<ul>
  <li>Trained machine learning models (Gradient Boosting, Random Forest, Stacked Ensemble)</li>
  <li>Experimental dataset and analysis</li>
  <li>Feature importance and SHAP analysis scripts</li>
  <li>Comparison with the proposed empirical equation and ACI predictions</li>
</ul>

<p>
<em>The repository will be fully updated and documented upon paper acceptance.</em>
</p>
